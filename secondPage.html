Question-3: What is event delegation in the DOM?
Answer: Event delegation in the DOM is a JavaScript technique where you attach a single event listener to a parent element, allowing it to handle events triggered on its child elements, instead of adding separate listeners to each child element; this leverages the "event bubbling" mechanism where events triggered on a child element propagate up to its ancestors in the DOM tree, making it efficient for handling events on large sets of dynamically added elements.
